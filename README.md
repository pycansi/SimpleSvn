# SimpleSvn
a simple svn wrapped. Supports directly viewing files, creating new folders, uploading files, downloading files, and deleting files on the svn server.

# dependencies
To rely on the [svn command-line tool](https://subversion.apache.org/download.cgi#recommended-release), you need to first configure the svn tool address, svn server address, download path (where to save the downloaded file), and terminal character encoding in the config.toml file.
